N2 is a lightweight CMS framework for ASP.NET that helps you build great web 
sites that anyone can update.

Major Changes
Functionality wise this release brings a login and user registration parts and 
improved installer. The primary changes include project rearrangements based on 
feedback and pain reports and a much easier configuration.

First time with N2?
* Download template implementation (below). Run in visual Studio.
* Look at the documentation
* Try a working implementation

Upgrading?
* Create a copy of the site
* Drop the compiled runtime
* Read upgrade_1.3.5.txt

Enjoy!
